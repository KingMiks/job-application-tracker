# Job Application Tracker (V1)

A desktop application built with **Python** and **Tkinter** that helps users organize and track job applications throughout the job search process.

This project was built to practice desktop application development, CRUD (Create, Read, Update, Delete) operations, JSON data persistence, and Git/GitHub workflow.

---

## Features

- Add new job applications
- Update existing job applications
- Delete one or multiple job applications
- Automatically save data to a local JSON file
- Automatically load saved applications when the program starts
- Input validation for required fields
- User-friendly error messages
- Persistent local storage using JSON

## How to Run

### Clone the repository

```bash
git clone https://github.com/KingMiks/job-application-tracker.git
```

### Navigate into the project

```bash
cd job-application-tracker
```

### Run the application

```bash
python main.py
```
